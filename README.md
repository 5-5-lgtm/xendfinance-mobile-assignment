# XendFinance Mobile Assignment

A cross-platform mobile application built with **React Native** and **Expo Router** as part of the technical assessment.

## 📱 Features
* **Authentication Flow:** Includes Welcome and Login screens.
* **Tab Navigation:** 5-tab layout featuring Home, Wallet, Referral, Plans, and More.
* **Wallet Management:** Detailed asset portfolio view showing CNGN, USDT, and USDC balances.
* **Clean UI:** Professional dark-themed interface with customized quick actions.

## 🛠️ Tech Stack
* **Framework:** React Native / Expo.
* **Routing:** Expo Router (File-based routing).
* **Language:** TypeScript.

## 📸 Screenshots
You can find visual proof of the running application in the `/app/screenshots` directory.

## 🚀 How to Run Locally
1. Clone the repository:
   `git clone https://github.com/5-5-lgtm/xendfinance-mobile-assignment.git`
2. Install dependencies:
   `npm install`
3. Start the project:
   `npx expo start`
